## About

This is my personal developer portfolio — a single-page application designed to present my projects, technical skills and professional background in a clean, modern interface. Fully responsive and available in both English and Spanish.

---

## Features

- **Responsive Design** — Optimized layout for desktop, tablet and mobile devices.
- **Bilingual (EN / ES)** — Full language toggle powered by i18next with browser language detection.
- **Project Showcase** — Featured project spotlight and additional project cards with links to live demos and repositories.
- **Case Studies** — Dedicated pages for in-depth project breakdowns.
- **Downloadable Resume** — PDF resume accessible directly from the site.
- **Contact Form** — Validated contact form with toast notifications.
- **Animations** — Scroll-reveal animations, typewriter effect and command rain visual.
- **Dark / Light Theme** — Theme toggle with system preference detection.
- **Tech Stack Display** — Visual representation of technologies and tools.

## Project Structure

```
src/
├── assets/           # Static assets (images, CV PDF)
├── components/       # Reusable UI components (shadcn/ui) and layout pieces
│   ├── layout/       # Navbar, Footer, LangToggle, ThemeToggle
│   └── ui/           # Base UI primitives
├── data/             # Static data (projects, certifications, techs, links)
├── features/         # Feature modules (About, Hero, Contact, Projects, etc.)
├── hooks/            # Custom hooks (theme, language, typewriter, reveal)
├── i18n/             # Internationalization config
├── lib/              # Utilities and error handling
├── routes/           # TanStack Router file-based routes
└── types/            # TypeScript type definitions
```

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (recommended) or Node.js 18+

### Installation

```bash
git clone https://github.com/wyn-code/Portafolio_2.0.git
cd Portafolio_2.0
bun install
```

### Development

```bash
bun run dev
```

### Build

```bash
bun run build
```
